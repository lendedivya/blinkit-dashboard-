# blinkit-dashboard-
Blinkit Sales &amp; Operations Dashboard (Power BI)
This project focuses on analyzing Blinkit (Instant Grocery Delivery) data using Power BI to gain insights into sales performance, customer behavior, product demand, and operational efficiency.
The dashboard is designed to help business stakeholders make data-driven decisions by visualizing key performance indicators (KPIs) in an interactive and easy-to-understand format.
